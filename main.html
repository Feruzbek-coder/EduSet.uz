🚀 Ishga Tushirish Qo'llanmasi

1️⃣ Python Kutubxonalarini O'rnatish

Terminal (PowerShell) da quyidagi buyruqni bajaring:

pip install -r requirements.txt

2️⃣ Flask Serverini Ishga Tushirish

Quyidagi buyruqni bajaring:

python app.py

Server ishga tushgach, konsolda "Running on http://127.0.0.1:5000" degan xabar ko'rasiz.

3️⃣ Brauzerda Ochish

Server ishga tushgandan so'ng, quyidagi manzilni brauzerde oching:

http://localhost:5000
